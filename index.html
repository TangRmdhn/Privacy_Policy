Kebijakan Privasi untuk SyncJadwal

Tanggal Efektif: 17 Oktober 2025

1. Pengenalan

Aplikasi web SyncJadwal ("Aplikasi") dibuat untuk membantu mahasiswa UPN "Veteran" Yogyakarta menyinkronkan jadwal kuliah mereka dari portal BIMA ke Google Calendar pribadi. Kebijakan Privasi ini menjelaskan bagaimana kami mengakses dan menggunakan data Anda.

2. Data yang Kami Akses

Saat Anda menghubungkan akun Google Anda, Aplikasi akan meminta izin untuk mengakses Google Calendar Anda dengan scope "https://www.googleapis.com/auth/calendar". Izin ini memungkinkan Aplikasi untuk:

  • Membuat event baru di kalender utama Anda (untuk jadwal kuliah).
  • Membaca event yang ada untuk menghindari duplikasi (jika diperlukan di masa depan).

3. Bagaimana Kami Menggunakan Data Anda

Data Google Calendar Anda hanya digunakan untuk satu tujuan: **membuat dan menempatkan jadwal kuliah yang Anda berikan ke dalam kalender Anda.** Kami tidak melakukan hal lain.

4. Berbagi dan Penyimpanan Data

Kami TIDAK PERNAH menyimpan, membagikan, atau menjual data Google Calendar Anda kepada pihak ketiga mana pun. Semua proses terjadi secara langsung antara browser Anda, server kami, dan Google API. Token autentikasi Anda hanya digunakan selama proses sinkronisasi dan tidak disimpan secara permanen.

5. Kepatuhan Kebijakan Google

Penggunaan dan transfer informasi yang diterima dari Google API oleh SyncJadwal akan mematuhi Google API Services User Data Policy, termasuk persyaratan Penggunaan Terbatas (Limited Use requirements).
